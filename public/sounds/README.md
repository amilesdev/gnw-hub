# GNW Play — Audio assets

Drop the MP3 files listed below into this folder. They're loaded lazily by
`lib/play/audio.ts`; until a file exists the game runs silently for that cue
(no errors). Filenames must match exactly (lowercase, `.mp3`).

See `Docs/GNW-Play-Audio-Manifest.md` for descriptions of each sound.

Required files:
- lobby-music.mp3
- celebration-music.mp3
- podium-land.mp3
- countdown-tick.mp3
- countdown-final.mp3
- answer-correct.mp3
- answer-wrong.mp3
- elimination.mp3
- heart-lost.mp3
- round-end.mp3
- game-start.mp3
